# Developer Docs

This directory contains the developer-facing documentation for Readalizer.

Start here if you are contributing or integrating Readalizer into another project.

## Index

- `docs/DEVELOPMENT.md` Local setup, workflows, and repository conventions.
- `docs/ARCHITECTURE.md` System overview and key components.
- `docs/CLI.md` CLI usage, options, and output behavior.
- `docs/CONFIGURATION.md` Config file reference and precedence rules.
- `docs/RULES.md` Writing rules and rulesets.
- `docs/SUPPRESSION.md` Attribute and inline suppression details.
- `docs/PARALLEL.md` Parallel execution model and worker internals.
- `docs/TROUBLESHOOTING.md` Common issues and diagnostics.
